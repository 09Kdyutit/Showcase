# Community Posts

Templates for Discord communities and Slack groups where Showcase can be shared authentically.  
Always check channel rules. Always be a genuine member before posting.

---

## Discord channels to target

- **Designer communities:** Design friends, Figma Discord, Superteam
- **Developer communities:** The Programmer's Hangout, Reactiflux, various bootcamp Discord servers
- **Job hunting / career:** CS Career Hub, Design career Discords
- **Indie maker / builder:** Indie Hackers Discord, ShipFast community
- **Startup founders:** Various YC alumni Discord servers (if you have access)

## Engagement rules for communities

1. Be a member for at least a few days before posting
2. Don't post the same thing in multiple servers on the same day
3. Always engage with replies — don't just drop a post and leave
4. If the server has a specific self-promo channel, use it
5. Don't spam the same message in multiple channels within the same server
